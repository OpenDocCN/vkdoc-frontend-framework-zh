## 下载

### Docker

```
docker pull apachecn0/vkdoc-jquery-zh
docker run -tid -p <port>:80 apachecn0/vkdoc-jquery-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g vkdoc-jquery-zh
vkdoc-jquery-zh <port>
# 访问 http://localhost:{port} 查看文档
```
